# Math Solver App

Образовательная веб-платформа для решения математических задач, разработанная в рамках выпускной квалификационной работы.
Посмотреть сайт можно здесь: http://72.56.11.167
## Описание проекта

Math Solver App представляет собой веб-приложение для изучения и тренировки математических навыков. Платформа позволяет пользователям решать задачи по различным разделам математики, получать автоматическую проверку ответов, просматривать подробные решения и отслеживать собственный прогресс.

Основной целью проекта является создание удобной интерактивной среды для самостоятельного изучения математики с использованием современных веб-технологий.

## Функциональные возможности

### Для пользователей

* регистрация и авторизация пользователей;
* решение математических задач;
* автоматическая проверка ответов;
* просмотр правильного решения;
* генерация новых вариантов задач;
* отслеживание статистики и прогресса обучения;
* доступ к справочным материалам.

### Поддерживаемые темы

#### Производные

* производная суммы функций;
* производная произведения функций;
* производная сложной функции.

#### Матрицы

* вычисление определителя;
* нахождение обратной матрицы;
* сложение матриц;
* умножение матриц.

#### Интегралы

* неопределённые интегралы;
* определённые интегралы.

#### Пределы

* пределы с использованием подстановки;
* пределы полиномиальных функций.

## Архитектура приложения

Приложение построено по архитектурному шаблону MVC (Model–View–Controller).

### Основные сущности

* User — пользователи системы;
* Topic — разделы математики;
* Subtopic — подразделы тем;
* Problem — задачи;
* Attempt — результаты решений пользователей;
* ReferenceMaterial — справочные материалы.

## Алгоритм работы

1. Пользователь выбирает тему и тип задачи.
2. Сервис генерации формирует новую задачу.
3. Задача отображается на странице решения.
4. Пользователь вводит ответ.
5. Сервис проверки анализирует корректность ответа.
6. Результат проверки сохраняется в истории пользователя.
7. Пользователь может просмотреть решение или сгенерировать новую задачу.

## Проверка ответов

Для проверки математических выражений используется специальный сервис проверки ответов.

Особенности проверки:

* поддержка математических выражений;
* сравнение числовых значений с учётом погрешности;
* обработка различных форматов ввода;
* проверка математической эквивалентности выражений.

## Установка и запуск

### Локальный запуск

Клонирование репозитория:

```bash
git clone <repository_url>
cd math_solver_app
```

Установка зависимостей:

```bash
bundle install
```

Настройка базы данных:

```bash
rails db:create
rails db:migrate
```

Запуск сервера:

```bash
bin/dev
```

После запуска приложение будет доступно по адресу:

```text
http://localhost:3000
```

## Запуск через Docker

Сборка контейнера:

```bash
docker build -t math_solver_app .
```

Запуск контейнера:

```bash
docker run -p 3000:3000 math_solver_app
```
